# RAG Seminar

This repository contains the slides for a research seminar on **Retrieval-Augmented Generation (RAG)**.

The seminar was presented at the **Machine Learning & Bioinformatics Lab, Pusan National University (PNU)** in 2026.

## Presenter
**Jiwon Lee**

## Description
This seminar introduces the fundamental concepts and recent developments in **Retrieval-Augmented Generation (RAG)**, a framework that combines large language models with external knowledge retrieval to improve factual accuracy and domain adaptability.
